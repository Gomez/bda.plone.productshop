=========================
bda.plone.shopviews
=========================

Views for bda.plone.shop

Views
============

-   @@products_view
-   @@productlist_view
-   @@productlisting_view
-   @@quicksand_view
-   @@productboxes_view
-   @@colorboxes_view
-   @@product_view


The views are intended as a starting point for your own views.
